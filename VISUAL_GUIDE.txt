 PART 1: HOW TO GET STARTED (First 30 Minutes)
═══════════════════════════════════════════════════════════════════════════════════

STEP 1: SET ENVIRONMENT VARIABLES (5 MINUTES)
┌─────────────────────────────────────────────────────────────────────────────┐
│ 1. Click ⚙️ icon (gear) in top-right corner                                 │
│ 2. Click "Vars" tab                                                         │
│ 3. Add all 11 variables:                                                    │
│                                                                             │
│   MONGODB_URI ...................... mongodb+srv://user:pass@cluster...    │
│   JWT_SECRET ....................... random-secret-key-from-openssl       │
│   BCRYPT_ROUNDS .................... 10                                    │
│   EMAIL_USER ....................... your-email@gmail.com                 │
│   EMAIL_PASSWORD ................... gmail-app-password                   │
│   RAZORPAY_KEY_ID .................. rzp_test_xxxxx                       │
│   RAZORPAY_KEY_SECRET .............. rzp_test_secret_xxxxx                │
│   CLOUDINARY_NAME .................. your-cloudinary-name                 │
│   CLOUDINARY_API_KEY ............... cloudinary-api-key                   │
│   CLOUDINARY_API_SECRET ............ cloudinary-api-secret                │
│   NEXT_PUBLIC_API_URL .............. http://localhost:3000                │
│                                                                             │
│ 4. Click "Save"                                                             │
└─────────────────────────────────────────────────────────────────────────────┘


STEP 2: CREATE SUPER ADMIN IN DATABASE (3 MINUTES)
┌─────────────────────────────────────────────────────────────────────────────┐
│ 1. Open MongoDB Atlas (https://www.mongodb.com/cloud/atlas)                │
│ 2. Click your cluster → Browse Collections                                  │
│ 3. Find 'users' collection → Click "+ INSERT DOCUMENT"                     │
│ 4. Paste this JSON (replace password field if needed):                     │
│                                                                             │
│ {                                                                           │
│   "email": "superadmin@ziyacreations.com",                                │
│   "password": "$2b$10$EIXwL/glR3e.tIUYKgfcLOLwKlY8aQrVVsRHCHzK...",       │
│   "name": "Super Admin",                                                   │
│   "role": "super_admin",                                                   │
│   "phone": "9876543210",                                                   │
│   "isEmailVerified": true,                                                 │
│   "createdAt": new Date(),                                                 │
│   "updatedAt": new Date()                                                  │
│ }                                                                           │
│                                                                             │
│ 5. Click "Insert Document"                                                 │
│ 6. ✅ Super Admin created!                                                 │
│                                                                             │
│ Login Credentials:                                                          │
│   Email: superadmin@ziyacreations.com                                     │
│   Password: superadmin123                                                  │
└─────────────────────────────────────────────────────────────────────────────┘


STEP 3: START YOUR APP (2 MINUTES)
┌─────────────────────────────────────────────────────────────────────────────┐
│ Open Terminal and run:                                                      │
│                                                                             │
│   $ cd /vercel/share/v0-project                                            │
│   $ pnpm dev                                                               │
│                                                                             │
│ Wait for:                                                                   │
│   ✓ Compiled successfully                                                   │
│   Ready on http://localhost:3000                                            │
│                                                                             │
│ Your app is running! 🚀                                                     │
└─────────────────────────────────────────────────────────────────────────────┘
